# My App – Feedback & Issue Tracker

Welcome! 👋  

This repository is the **public place to report bugs, request features, and share feedback** for **My App**.

The main source code repository for the application is private, but we want to make it easy for users to help improve the project.

---

## 🐞 Report a Bug

If something isn’t working as expected, please open a bug report.

Before submitting:
- Check if the issue has already been reported
- Include clear steps to reproduce the problem
- Add screenshots if possible

👉 **Create a bug report:**  
https://github.com/ORG/REPO/issues/new?template=bug_report.yml

---

## 💡 Request a Feature

Have an idea to improve the app? We'd love to hear it.

👉 **Submit a feature request:**  
https://github.com/ORG/REPO/issues/new?template=feature_request.yml

Please include:
- A clear description of the idea
- Why it would be useful
- Any alternative solutions you considered

---

## 💬 Not Sure?

If you're unsure whether something is a bug or a feature request, feel free to open an issue anyway. We’ll help categorize it.

---

## 🙏 Community Guidelines

Please try to:

- Be respectful and constructive
- Keep issues focused on a single topic
- Provide as much detail as possible
- Search existing issues before opening a new one

---

## 🔒 About the Code

The main application repository is private.

However:
- All **bug reports**
- All **feature discussions**
- All **user feedback**

are handled here.
